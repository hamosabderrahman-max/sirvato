Keyboard Shortcuts

Complete reference for all keyboard shortcuts

Global Shortcuts

ShortcutAction
Alt + TToggle dark / light theme
Alt + LToggle language (EN ↔ IT)
Alt + HGo to Homepage
Alt + DOpen Live Demo
EscClose any open modal or overlay

Demo Page Shortcuts

ShortcutAction
Ctrl + EnterGenerate prompt from current inputs
Ctrl + Shift + CCopy generated prompt to clipboard
Ctrl + RReset / clear all input fields
TabMove to next input field
Shift + TabMove to previous input field

Code Block Shortcuts

ShortcutAction
Click Copy buttonCopy code block to clipboard
Ctrl + A in codeSelect all code in block

Navigation Shortcuts

ShortcutAction
Alt + ←Navigate back
Alt + →Navigate forward
Ctrl + FBrowser search on page
HomeScroll to top of page
EndScroll to bottom of page
