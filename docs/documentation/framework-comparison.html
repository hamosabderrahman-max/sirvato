Framework Complexity Explained

Beginner Tier

ICDF Engine and AI Trigger Protocol. Minimal cognitive overhead. Learn these first.

Complexity: ★★☆ or lower

Intermediate Tier

MICRO Layer, RCR-EOC, and basic AUTO-FLOW. More components but each has a clear role.

Complexity: ★★★

Advanced Tier

AUTO-FLOW PRO, AUTO-FLOW REACTIVE. Requires understanding of conditional logic and iteration.

Complexity: ●●●●

Expert Tier

Unified Mode. All 11 components. Reserved for high-stakes, enterprise-grade work only.

Complexity: ●●●●●
