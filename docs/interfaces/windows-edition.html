Windows Edition

Familiar Windows-style interface bringing SIRVATO's full power to desktop-paradigm users. All the capability of SIRVATO OS in a UI pattern you already know.

Familiar UI Taskbar Navigation Window Management

What is Windows Edition?

Windows Edition bridges the gap between SIRVATO OS's powerful cloud-native capabilities and the desktop computing paradigm that billions of people already know. It delivers the full SIRVATO feature set through a familiar windowed desktop experience — complete with taskbar, Start-style menu, and classic window chrome.

For teams transitioning from traditional IT management tools, Windows Edition dramatically reduces the learning curve without compromising on the depth and power of the underlying platform. Drag windows, pin shortcuts, right-click for context menus — it's all there.

Familiar Power

Windows Edition proves that user-friendly and enterprise-powerful are not opposites. New team members are productive within minutes; experienced administrators discover depth over weeks.

Core Features

Feature 01
🖥️
Familiar UI Paradigm

Desktop icons, window chrome, right-click menus, and taskbar navigation. Zero retraining required for your team.

Feature 02
📌
Taskbar Navigation

Pin your most-used tools to the taskbar for one-click access. Running apps show live status indicators.

🪟
Window Management

Minimize, maximize, resize, snap-to-half. Full window management including multi-monitor support and virtual desktops.

Create desktop shortcuts to any SIRVATO view, dashboard, or action. Double-click to open instantly.

Configuration

YAML 
# sirvato.yaml — Windows Edition Interface
interfaces:
  windows_edition:
    enabled: true
    desktop:
      wallpaper: "sirvato-dark"
      icon_size: "medium"
    taskbar:
      position: "bottom"
      pinned:
        - "metrics"
        - "deployments"
        - "security"
        - "logs"
    windows:
      snap_enabled: true
      virtual_desktops: 4
      multi_monitor: true
    start_menu:
      enabled: true
      search_enabled: true
