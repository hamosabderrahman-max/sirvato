FULLSCREEN MODE

Immersive full-canvas interface for deep-focus operations and system visualization. Every pixel of your display dedicated to SIRVATO OS.

Full Viewport Distraction-Free Focus Mode

What is Fullscreen Mode?

Fullscreen Mode takes over the entire display surface, removing all system chrome, browser UI, and OS elements to give SIRVATO OS maximum visual real estate. It's the interface of choice for NOC displays, dedicated monitoring stations, and deep-focus operational sessions.

In Fullscreen Mode, every view — charts, logs, maps, dashboards — scales to occupy the full canvas. Keyboard-first navigation means your hands never leave the keys. A ghost nav bar fades in on cursor movement and disappears within seconds to keep the focus on your data.

Zero Distraction, Maximum Clarity

Fullscreen Mode is built for the hours when nothing else matters except understanding what your system is doing right now. It removes every visual distraction so the signal is all that remains.

Core Features

Feature 01
⬛
Full Viewport Coverage

Acquires 100% of the display surface. No browser bars, no OS taskbars, no chrome — just your system data at maximum scale.

Feature 02
🎯
Distraction-Free

Ghost navigation fades in only when needed and auto-hides in 3 seconds. UI chrome is invisible unless you summon it.

Feature 03
⌨️
Keyboard Shortcuts

Navigate every view, switch dashboards, trigger actions — all via keyboard. Mouse is optional in Fullscreen Mode.

Feature 04
🔭
Focus Mode

Deep-ops, incident-response, and monitoring presets tune the display for your current operational context automatically.

Configuration

YAML 
# sirvato.yaml — Fullscreen Mode Interface
interfaces:
  fullscreen_mode:
    enabled: true
    trigger_key: "F11"
    ghost_nav:
      enabled: true
      hide_after: "3s"
      trigger: "mouse-top"
    focus_presets:
      - name: "deep-ops"
        panels: ["metrics", "logs"]
      - name: "incident-response"
        panels: ["alerts", "logs", "deploy"]
      - name: "monitoring"
        panels: ["metrics", "network", "security"]
    multi_monitor: true
    refresh_rate: "250ms"
