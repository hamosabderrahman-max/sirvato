What is Micro Sidebar?

Micro Sidebar is the efficiency-maximizing interface layer for SIRVATO OS. It replaces wide navigation panels with a slender, icon-first sidebar that sits at the edge of your workspace — present but unobtrusive, providing access to every feature with a single click or hover expand.

The sidebar is context-aware: it recognizes what you're working on and surfaces relevant tools at the top of the stack. Editing a deployment config? The Deployment Manager, Config Editor, and Log Stream float to the top automatically. Responding to an alert? Alert Hub and Security Center lead.

Maximum Workspace, Full Control

Micro Sidebar was built for developers who work in dense, multi-pane environments — code editors, split terminals, dashboards — where every pixel of horizontal space is precious, but operational control must remain within arm's reach.

Core Features

Feature 01
📐
Compact Design

48px collapsed width. Expands to 220px on hover with a 200ms delay to avoid accidental triggers. Reclaims maximum workspace.

Feature 02
🔑
Full Access

All SIRVATO OS tools and modules remain one click away. Compact does not mean limited — every feature is reachable instantly.

Feature 03
🗂️
Collapsible

Collapse entirely to an icon strip or fully expand to a rich nav panel. Pin the state you prefer, or let it auto-manage.

Feature 04
🎯
Context-Aware

Intelligently surfaces the most relevant tools based on your current activity. The sidebar adapts to your workflow automatically.

Configuration

YAML 
# sirvato.yaml — Micro Sidebar Interface
interfaces:
  micro_sidebar:
    enabled: true
    position: "left"
    width:
      collapsed: 48
      expanded: 220
    expand_trigger: "hover"
    expand_delay: "200ms"
    context_aware: true
    pinned_tools:
      - "metrics"
      - "logs"
      - "deployments"
      - "security"
      - "alerts"
    context_suggestions: true
    max_suggestions: 3
    icon_set: "compact-v3"
