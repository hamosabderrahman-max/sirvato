MICRO Sidebar

5-Panel Expandable MICRO Sidebar System

An expandable sidebar system with 5 dedicated panels — one for each MICRO component. Real-time prompt generation as you fill each panel. The definitive interface for the MICRO framework specialist.

Overview

What is MICRO Sidebar?

MICRO Sidebar is the purpose-built interface for the MICRO framework — M-I²-C-R-O — the communication architecture at the heart of SIRVATO OS. Instead of a general-purpose prompt editor, MICRO Sidebar presents exactly five panels, one for each MICRO component. Message. Intention². Context. Rhythm. Output. Each panel is labelled, described, and guided with contextual hints. As you fill each panel, the live preview area below assembles the complete MICRO prompt in real time.

The sidebar metaphor is intentional. MICRO Sidebar is designed to live alongside your primary work — whatever that work may be. Collapse it to a slim 5-button vertical strip when you need screen space. Expand any single panel to full width when you need to think deeply about that component. The workflow is: open the Message panel, articulate your core message, close it, open Intention², define your dual-layer intent, and proceed through the components until the preview shows a complete, valid MICRO structure ready to deploy.

Features

Core Capabilities

✦

5 Expandable Panels (M, I², C, R, O)

Each MICRO component occupies its own dedicated panel. The panel header shows the component letter, full name, and a one-line description of what belongs in that component. Click the header to expand the panel to an editable text area. Click again to collapse it back to the single-letter tab. Multiple panels can be open simultaneously for cross-referencing as you write.

✦

Real-Time Prompt Preview as You Type

The preview area at the bottom of the sidebar updates keystroke-by-keystroke as you type in any panel. The preview renders the complete MICRO structure with proper labels and formatting — exactly what will be sent to the AI. Completeness indicators (●●●○○ dots) show which components have been filled and which are still empty, giving you an at-a-glance prompt quality signal.

✦

Smooth Panel Expand/Collapse Animations

Panel transitions use CSS spring animations at 300ms — fast enough to feel responsive, slow enough to communicate spatial position clearly. The animation preserves orientation so you always know which component you are looking at relative to the full M-I²-C-R-O stack. The animation can be reduced-motion disabled for accessibility compliance.

✦

Collapsible to Single-Letter Tabs

When you need the full screen for other work, collapse the entire sidebar to a slim strip of 5 single-letter tabs: M, I², C, R, O. Each tab is colour-coded — filled tabs appear in the active accent colour, empty tabs in muted grey. The entire sidebar width in collapsed mode is 32px, taking up less than 2% of a standard 1920px monitor width.

✦

Panel Linking — Context Flows Between Panels

Enable Panel Linking mode and the sidebar automatically propagates context. If you specify an audience in the Context panel, the Rhythm panel surfaces tone suggestions calibrated to that audience. If your Message contains a specific topic, the Intention² panel suggests strategic objectives aligned with that topic. Panel Linking transforms the sidebar from a form into a guided composition system.

✦

Auto-Complete Suggestions Per Panel

Each panel has a context-aware auto-complete system trained on MICRO framework best practices. The Message panel suggests clarifying the core proposition. Intention² auto-complete proposes dual-layer intent framings. Context suggests relevant environmental variables. Rhythm offers tone and cadence descriptors. Output proposes format specifications. Accept any suggestion with Tab; reject and keep typing to dismiss.

✦

Export Complete MICRO Structure as JSON

Click Export in the preview panel to generate a structured JSON representation of your complete MICRO prompt. The JSON includes all five component values, metadata (timestamp, completion status, word counts per component), and a formatted_prompt key containing the complete assembled prompt as a string. The JSON format integrates directly with SIRVATO OS API endpoints and Master Block import.

Usage Example

MICRO Sidebar in Action

Open MICRO Sidebar and expand the M (Message) panel first. Articulate the single core message you want to communicate. Move through I², C, R, and O in sequence. The preview builds as you go. The following shows a complete MICRO structure built in the sidebar for a product launch announcement:

MICRO Sidebar — Complete Structure Export 
{
  "framework": "MICRO",
  "components": {
    "M": "Our new developer API is now in public beta — it is ready to use today.",
    "I2": {
      "stated": "Inform developers that the API is available",
      "strategic": "Create urgency and excitement to drive immediate sign-ups and early adoption momentum"
    },
    "C": "Tech developer community, primarily backend engineers, active on Hacker News and GitHub",
    "R": "High energy, peer-to-peer tone — developer to developer, not company to customer. Direct, no fluff.",
    "O": "Tweet thread: 5 tweets, max 280 characters each, first tweet must hook in < 10 words"
  },
  "metadata": {
    "timestamp": "2024-01-15T14:23:01Z",
    "completion": "5/5",
    "total_words": 94
  },
  "formatted_prompt": "M: Our new developer API is now in public beta — ready to use today.\nI²: [stated] Inform developers API is available / [strategic] Drive urgency and early adoption\nC: Tech developers, backend engineers, Hacker News + GitHub community\nR: High energy, peer-to-peer, developer to developer, direct, no fluff\nO: Tweet thread, 5 tweets, max 280 chars, first tweet hooks in under 10 words"
}

Best for: MICRO framework specialists, structured communication work across marketing, strategy, and product domains, mobile-friendly prompting where screen space is limited, and users who prefer a guided component-by-component workflow over an open freeform editor.
