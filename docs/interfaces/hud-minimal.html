HUD Minimal

Streamlined Heads-Up Display Interface

The most efficient interface in SIRVATO OS. Stripped to essentials — only what you need, exactly when you need it. Zero cognitive overhead, maximum throughput.

Overview

What is HUD Minimal?

HUD Minimal is the flagship stripped-down interface of SIRVATO OS, designed around a single philosophical principle: remove everything that does not serve the prompt. Inspired by aerospace Heads-Up Displays where pilots receive only the critical information they need at any given moment, HUD Minimal surfaces framework indicators, token counts, and status signals in a compact two-row chrome — then steps out of the way so you can work.

The result is a sub-100ms cognitive load interface. Experienced prompt engineers report up to 40% faster iteration cycles when switching to HUD Minimal from richer UI environments. Every element earns its place through usefulness, and everything else is removed entirely.

Features

Core Capabilities

✦

Single-Task Focus

The interface enforces one active prompt at a time. No tabs, no history panels, no sidebar distractions. Your current task occupies the entire interaction area, eliminating context-switching cost and keeping your attention locked on the work at hand.

✦

Real-Time Token Counter

A live token count updates with every keystroke. Always know your budget before you send. Colour coding shifts from green to amber to red as you approach model context limits, giving you an instant visual signal without interrupting your flow.

✦

Framework Indicator Badge

A compact badge in the lower chrome shows the active SIRVATO OS framework — ICDF, MICRO, RCR-EOC, AUTO-FLOW, or Unified. Switch frameworks with a single keystroke. The badge updates instantly, keeping you oriented without requiring a sidebar or menu.

✦

Status LED

Three states — Ready (green), Processing (amber pulse), Complete (solid blue) — communicate system state at a glance. The LED is designed to be readable in peripheral vision so you never need to look away from your prompt text to check system status.

✦

One-Click Prompt Copy

A floating copy button appears on hover over the active prompt area. One click copies the complete, framework-formatted prompt to your clipboard in the correct structure for the active framework. No selection required, no formatting lost.

✦

Keyboard Shortcut Support

Full keyboard navigation. ⌘K activates the interface, ⌘Enter submits, ⌘C copies the formatted prompt, ⌘F cycles frameworks, and Escape clears the active task. Experienced users never touch a mouse while working in HUD Minimal.

✦

Minimal 2-Line Input Area

The input area starts at 2 lines and auto-expands as you type, never exceeding 12 lines before entering scroll mode. This keeps the interface compact for short prompts while gracefully accommodating longer structured inputs without layout shifts.

Usage Example

HUD Minimal in Action

Activate HUD Minimal with ⌘K. The interface initialises in under 200ms. Start typing immediately — the framework auto-detects structure and activates the appropriate SIRVATO OS engine. The following example shows an ICDF prompt in HUD Minimal mode:

HUD Minimal — ICDF Mode 
STATUS: ● READY  │  FRAMEWORK: ICDF  │  TOKENS: 0

> I: Rewrite this product description for a technical audience
  C: SaaS B2B platform, audience is CTOs and engineering leads
  D: [paste your product description here]
  F: 3 paragraphs, technical precision, no marketing fluff

STATUS: ● PROCESSING  │  TOKENS: 87  │  ⌘C to copy

Best for: Quick iterations, high-volume prompting sessions, experienced users who have internalised framework structures and need minimal scaffolding to work at speed.
