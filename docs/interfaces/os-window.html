OS Window

Multi-panel windowed interface for concurrent management of multiple system views. Run any combination of panels simultaneously with full drag, resize, and state persistence.

Multi-Panel Drag & Drop State Persistence

What is OS Window?

OS Window brings the productivity of a true windowed environment to SIRVATO OS management. Open multiple panels — logs, metrics, deployments, configs — arrange them freely, and have them all updating in real time simultaneously.

Each window is independently scrollable, filterable, and resizable. Window layouts are saved automatically per user session, so your preferred workspace is always ready when you return. Multi-monitor support lets you span panels across displays for maximum coverage.

Spatial Thinking

OS Window is designed for engineers who think spatially. By letting you arrange panels according to how you reason about your system, it dramatically reduces the cognitive load of managing complex, multi-layered infrastructure.

Core Features

Feature 01
🪟
Multi-Panel

Open unlimited panels simultaneously. Logs, metrics, deployments, configs — all visible and live-updating at once.

Feature 02
✋
Drag & Drop

Freely reposition and rearrange panels by dragging. Snap-to-grid assistance makes layout management effortless.

Feature 03
↔️
Resizable Panes

Drag panel edges to resize. Panels flow and reflow in response to your preferred proportions with fluid animation.

Feature 04
💾
State Persistence

Layouts are auto-saved locally and synced to the cloud. Resume exactly where you left off on any device.

Configuration

YAML 
# sirvato.yaml — OS Window Interface
interfaces:
  os_window:
    enabled: true
    default_layout: "my-workspace"
    panels:
      - id: "metrics"
        position: "top-left"
        size: "half"
      - id: "logs"
        position: "top-right"
        size: "half"
      - id: "deployments"
        position: "bottom-left"
        size: "half"
      - id: "alerts"
        position: "bottom-right"
        size: "half"
    persistence:
      local: true
      cloud_sync: true
    snap_to_grid: true
    multi_monitor: true
