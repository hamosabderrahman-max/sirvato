# SIRVATO OS — Official Website

This directory contains the complete official website for SIRVATO OS, the prompt engineering operating system.

## Structure

- `index.html` — Homepage
- `demo.html` — Interactive MICRO prompt builder
- `tools.html` — Framework selector tool
- `frameworks/` — 8 framework documentation pages
- `interfaces/` — 8 interface documentation pages
- `micro-supra/` — 6 MICRO SUPRA-TOTALITY pages
- `documentation/` — 7 reference documentation pages
- `css/` — Stylesheets (style, dark-mode, responsive, animations)
- `js/` — JavaScript (theme, i18n, main, animations, demo)

## Design

Dark cyberpunk OS aesthetic with neon accents on pure black background.
Primary: #00d4ff (cyan) | Secondary: #7c3aed (purple) | Accent: #00ff88 (green)

## Features

- Dark/light mode with localStorage persistence
- EN/IT language toggle
- Interactive MICRO prompt builder
- Framework selector tool
- Matrix rain canvas animation
- Scroll-reveal animations
- Mobile responsive with hamburger menu
- Copy-to-clipboard for all code blocks
