Choosing the Right Framework

Just getting started?

Use ICDF Engine — it is the simplest framework and handles most common use cases automatically via intent-driven decision routing.

Building microservices?

Use MICRO Framework — purpose-built for small, focused services with minimal overhead and AI-powered service mesh.

Enterprise scale?

Use AUTO-FLOW PRO or Unified Mode for full enterprise orchestration with compliance, audit trails, and multi-region support.

Event-driven systems?

Use AUTO-FLOW REACTIVE for systems that need to respond instantly to state changes and event streams with reactive pipelines.
