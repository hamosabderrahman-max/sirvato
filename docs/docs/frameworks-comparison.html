When to Use Each Framework

  • ICDFTechnical docs, code generation, structured analysis. You know exactly what you want.
  • MICROSocial media, email, presentations. Tone and rhythm are as important as content.
  • RCR-EOCResearch, UX work, complex writing. Multiple stakeholders and constraints involved.
  • Auto-FlowRepeatable workflows, campaigns, planning. Need a reliable, step-by-step process.
  • Auto-Flow PROProduct launches, executive communication, sales copy. High stakes, zero margin for error.
  • Unified ModeWhen you need the absolute best result and complexity justifies the effort.
