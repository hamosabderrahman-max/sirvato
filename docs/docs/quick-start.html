Quick Start

Get up and running with SIRVATO OS in under 5 minutes.

📖 Documentation

What is SIRVATO OS?

SIRVATO OS is a unified AI prompt operating system that combines multiple prompt engineering frameworks into a single, coherent ecosystem. It provides structured approaches to crafting precise, powerful prompts for any AI model.

Core Concept

Instead of using prompts ad-hoc, SIRVATO OS provides a systematic, repeatable process using proven frameworks: ICDF for structure, MICRO for rhythm, RCR-EOC for scenarios, and Auto-Flow for automation.

Step 1 — Choose Your Framework

  • ICDFFor precise, structured tasks where format matters.
  • MICROFor content creation where rhythm and tone are critical.
  • RCR-EOCFor complex scenarios with multiple constraints.
  • Auto-FlowFor systematic, step-by-step prompt construction.
  • Unified ModeFor maximum precision when stakes are high.

Step 2 — Fill the Fields

Each framework has specific fields that guide you through the prompt creation process. The key fields are:

TARGET    → What do you want to achieve?
ROLE      → Who should the AI be?
CONTEXT   → What is the situation?
REQUEST   → What exactly do you need?
OUTPUT    → What format should the result be?
CONSTRAINTS → What are the limits?

Step 3 — Generate and Iterate

Use the Interactive Demo to fill the fields and generate your prompt. Copy the output and paste it directly into your AI model of choice.

Pro tip: Start with Unified Mode for complex tasks — it activates all frameworks simultaneously for maximum precision.

Example: Your First SIRVATO Prompt

🌑 SIRVATO OS — UNIFIED PROMPT

⚡ TARGET   → Write a compelling product description
🟣 ROLE     → Senior copywriter, conversion specialist
