AI Trigger Protocol — Validation Rules

  • T1 CoherenceRole and Instruction must be semantically aligned. Expert role → expert-level instruction.
  • T2 RhythmRhythm field must be explicitly defined. "Normal" is not a valid rhythm specification.
  • T3 DataAll referenced data must be present. No hanging references like "as mentioned above".
  • T4 OutputOutput format must be specific. "A document" fails; "500-word markdown article with H2 sections" passes.
  • T5 ConstraintsAt least one explicit constraint must be present. Even "no constraints" must be stated.
