Auto-Flow Reactive

Adaptive cascade — each field activates the next for fully dynamic, context-aware prompt generation.

Components

  • TARGET Activates the cascade — defines the ultimate objective.
  • ROLE → MESSAGE Role informs the voice; message emerges from the role.
  • CONTEXT → INTENTION Context shapes intention; intention refines context.
  • REQUEST → RHYTHM Request determines the rhythm and energy needed.
  • EXAMPLES → OUTPUT Examples define what good looks like; output mirrors it.
  • CONSTRAINTS → ICDF Constraints feed into ICDF structure automatically.
  • TRIGGERS → UNIFIED → EXECUTE AI triggers validate; Unified Mode fuses; Execute fires.

Example

🌪️ AUTO-FLOW REACTIVE — CASCADE
────────────────────────────────────────
TARGET     → "Launch AI productivity app"
↓
ROLE       → "Tech entrepreneur, serial founder"
↓
MESSAGE    → "AI isn't replacing you — it's multiplying you"
↓
CONTEXT    → "ProductHunt launch day, 2k watchers"
↓
INTENTION  → "Drive signups + virality"
↓
REQUEST    → "Write the PH tagline + first comment"
↓
RHYTHM     → "Punchy, confident, founder voice"
↓
OUTPUT     → "Tagline (60 chars) + comment (200 words)"
↓
UNIFIED → EXECUTE ✅
────────────────────────────────────────
