Auto-Flow PRO

Guided workflow with micro-actions at every step: Specify · Refine · Reduce · Amplify.

Components

  • Specify Add precise details, numbers, names, formats.
  • Refine Remove ambiguity, sharpen focus, clarify intent.
  • Reduce Cut noise, remove redundancies, compress complexity.
  • Amplify Add power, emotional charge, authority or urgency.

Example

🔥 AUTO-FLOW PRO — GUIDED
────────────────────────────────────────
Each step includes micro-actions:

STEP: Write a product launch announcement

→ Specify:  "SaaS product, €49/mo, B2B, 50-seat teams"
→ Refine:   "Focus on ROI, not features"
→ Reduce:   "Max 3 key benefits, no jargon"
→ Amplify:  "Add social proof, urgency, FOMO"

OUTPUT: Precision-crafted announcement prompt
────────────────────────────────────────

Best Use Cases

  • Product launches
  • High-stakes content
  • Sales copy
  • Executive communication
