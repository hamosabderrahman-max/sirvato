ICDF Engine

Instruction · Context · Data · Format — structured prompt architecture for precision output.

Components

  • I — Instruction The specific action the AI must perform. Clear, imperative, unambiguous.
  • C — Context The scenario, environment, audience and constraints that frame the task.
  • D — Data Source material, facts, figures, examples or raw information to process.
  • F — Format The exact output form: list, table, email, JSON, markdown, code, essay…

Example

🟣 ICDF ENGINE
────────────────────────────────────────
I — Instruction: Write a technical blog post
C — Context:     Senior developers, Python ecosystem, 2025
D — Data:        Python 3.12 new features: type system improvements, perf gains
F — Format:      1500-word article, H2 sections, code examples, conclusion
────────────────────────────────────────
→ Ready to execute

Best Use Cases

  • Technical documentation
  • Content generation
  • Data analysis tasks
  • Code generation
