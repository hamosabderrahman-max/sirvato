MICRO Layer

Message · Intention² · Context · Rhythm · Output — rhythm-driven prompt construction for fluid AI interaction.

Components

  • M — Message The core message, topic or central idea to communicate.
  • I² — Intention The double intention: what to achieve AND the desired emotional/cognitive effect.
  • C — Context The situation, platform, audience and environmental factors.
  • R — Rhythm Tone, pace, energy, formality, density. How the content should feel.
  • O — Output The final form: post, email, script, analysis, report, code…

Example

🔵 MICRO LAYER
────────────────────────────────────────
M — Message:    AI is transforming how we work
I² — Intention: Inspire action + overcome fear of change
C — Context:    LinkedIn post, 8k followers, Monday morning
R — Rhythm:     Energetic, punchy, short paragraphs
O — Output:     3-paragraph post, hook + body + CTA
────────────────────────────────────────
→ Ready to execute

Best Use Cases

  • Social media content
  • Email marketing
  • Presentations
  • Brand messaging
