## Overview

MICRO is a 5-component rhythm-based framework for precise communication and message delivery.

---

## Framework Components

### M — Message

```
M — Message
→ [ Messaggio centrale da trasmettere ]
```

Define the core message or central concept you want to communicate.
The essential information or idea to be delivered.

---

### I² — Intention

```
I² — Intention
→ [ Effetto desiderato sul destinatario ]
```

Specify the desired effect or impact on the recipient.
What emotional or intellectual response do you want to trigger?

---

### C — Context

```
C — Context
→ [ Scenario, pubblico, situazione ]
```

Establish the scenario, target audience, and situational factors.
Who are you communicating with? What is the context?

---

### R — Rhythm

```
R — Rhythm
→ [ Tono, energia, intensità, voce ]
```

Set the tone, energy level, intensity, and voice of the communication.
How should the message be delivered? What rhythm should it follow?

---

### O — Output

```
O — Output
→ [ Forma finale: testo, lista, email, codice, ecc. ]
```

Define the final form: text, list, email, code, dashboard, etc.
What is the desired output format?

## Integration Points

MICRO is embedded throughout SIRVATO OS:

- **ICDF Engine** — Integrated as a separate communication layer
- **RCR-EOC + ICDF Unified** — Active in unified operation mode
- **AUTO-FLOW Frameworks** — Drives rhythm and output selection
- **Unified Mode (11 components)** — MICRO components map to Role, Message, Intention, Context, Rhythm, Output slots
- **FULLSCREEN TOTALITY** — Live canvas uses MICRO for structured prompt delivery
