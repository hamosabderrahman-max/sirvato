## ⚙️ Windows-Native Command Structure

### PowerShell Integration

```powershell
# Initialize SIRVATO OS Environment
$env:SIRVATO_HOME = "C:\Users\$env:USERNAME\sirvato"
$env:SIRVATO_BRANCH = "main"
$env:SIRVATO_REPO = "hamosabderrahman-max/sirvato"

# Load Framework
. "$env:SIRVATO_HOME\frameworks\auto-flow-reactive.ps1"

# Execute Unified Console
Invoke-SirvatoConsole -Mode "UNIFIED" -Framework "RCR-EOC+ICDF"
```

### Batch Command Support

```bat
@ECHO OFF
SET SIRVATO_HOME=%USERPROFILE%\sirvato
SET SIRVATO_BRANCH=main
SET SIRVATO_REPO=hamosabderrahman-max/sirvato

ECHO ================================================
ECHO  SIRVATO OS — Windows Edition
ECHO  Branch: %SIRVATO_BRANCH%
ECHO ================================================
CALL "%SIRVATO_HOME%\launch.bat"
```
